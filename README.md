## 📃 프로젝트 소개

**김영한님의 강의를 수강하고 배운내용을 토대로 스프링 MVC를 활용하여 게시판 서비스를 구현해보자 계획하였습니다.**

- 학습한 스프링의 기본 기능을 최대한 활용하여 게시판 서비스를 구현해보았습니다.

- DB 기술은 메모리부터 시작하여 Jdbc 부터 Jpa 까지 점진적으로 여러 기술을 적용해보았습니다.

<br>

## 🛠 개발 환경
- `Intellij 2023.2.3`
- `java 18`
- `Gradle 8.5`
- `Spring boot 3.2.2`

<br>

## 📁 프로젝트 기능

### [사용자]
- **회원가입 및 로그인**
    - 중복된 회원가입과 부적절한 사용자의 로그인 방지를 위한 중복 및 유효성 검사 수행
    
- **필터와 인터셉트**
    - 현재는 필터와 인터셉트를 활용하고 있으나, 보다 강력한 보안을 위해 Spring Security로 전환 계획

- **사용자의 역할관리**
    - 사용자의 역할을 나뉘어 관리하는 기능 추가
    - 관리자, 일반 사용자 등 다양한 역할 부여 및 권한 제어
  
- **사용자 회원수정**

### [게시판]
- **CURD 기능**
  - 게시물의 생성(Create), 조회(Read), 수정(Update), 삭제(Delete) 기능 제공

- **검색 기능 개선**
  - 게시물 제목 또는 내용을 기준으로 검색하는 기능

- **이미지 업로드 및 표시**
  - 게시물에 이미지 업로드 기능
    
  - 업로드된 이미지는 게시물 내에서 썸네일 형태로 표시
- **조회수 표시**
  - 게시글 조회수 표시 기능

- **페이징 기능**
